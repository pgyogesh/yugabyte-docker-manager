# Setting Up Yugabyte Logo Icon

To use the Yugabyte logo as the extension icon:

1. **Download the Yugabyte logo:**
   - Visit https://www.yugabyte.com/brand-assets/ or
   - Download from https://github.com/yugabyte/yugabyte-db (look for logo assets)

2. **Convert to PNG format:**
   - The icon should be 512x512 pixels
   - Save it as `assets/extension-icon.png`

3. **Replace the current icon:**
   - Replace `assets/extension-icon.png` with the Yugabyte logo
   - The icon will automatically be used by Raycast

**Note:** The icon file must be named `extension-icon.png` and placed in the `assets/` directory as specified in `package.json`.
